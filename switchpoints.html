扩充至两个转折点¶

之前的模型是我们假设有一个转折点，从而估计转折点在哪一天。但有没有可能不止一个转折点呢？下面我们将转折点个数进行扩充，假如有2个转折点。

接下来进行建模。和之前的几乎一样。

（1）短信数量符合泊松分布。泊松分布十分适合描述某段时间内，事件具体的发生概率。

$$ C_i \sim \text{Poisson}(\lambda) $$

（2）确定泊松分布参数$\lambda$的先验分布。

$$ \lambda = \begin{cases} \lambda_1 & \text{if } t \lt \tau_1 \cr \lambda_2 & \text{if } \tau_1 \le t \lt \tau_2 \cr \lambda_3 & \text{if } t \ge \tau_2 \end{cases} $$\begin{align} &\lambda_1 \sim \text{Exp}( \alpha ) \\\ &\lambda_2 \sim \text{Exp}( \alpha )\\\ &\lambda_3 \sim \text{Exp}( \alpha ) \end{align}

（3）确定$\tau_1$和$\tau_2$的先验

\begin{align} & \tau_1 \sim \text{DiscreteUniform(1,n_count_data-1) }\\\\ & \tau_2 \sim \text{DiscreteUniform($\tau_1$,n_count_data) } \end{align}

n_count_data 是总天数。两个转折点有大小之分，为了保证二者取到的值没有交叉（按照分段函数），$tau_2$至少有一个值可以取到。因此二者符合上述分布

将上述分布转换为代码

model=pm.Model()
with model:
    alpha = 1.0 / count_data.mean() #1/短信的平均值
    #随机变量的先验，这里三个先验一样
    lambda_1 = pm.Exponential("lambda_1", alpha)
    
    lambda_2 = pm.Exponential("lambda_2", alpha)
    
    lambda_3= pm.Exponential("lambda_3", alpha)
    
    tau1 = pm.DiscreteUniform("tau1", lower=0, upper=n_count_data-1)  
    
    tau2=pm.DiscreteUniform("tau2", lower=tau1, upper=n_count_data)

接下来需要得到$\lambda$

此处需注意，在原先的例子里使用了

$$lambda_ = pm.math.switch(tau >= idx, lambda_1, lambda_2)$$

一个类似于if语句的Theano函数。它使用第一个参数在后面的两个参数之间切换。而此处需要$if,elif,else$的结构完成分段函数。单独的$switch$语句是无法完成的。

In [3]:
with model:
    idx = np.arange(n_count_data)  # Index，当前天数
    
    lambda_temp = pm.math.switch(tau1 >= idx , lambda_1,lambda_2)

    lambda_ = pm.math.switch(tau2 >= idx,lambda_temp,lambda_3 )

    observation = pm.Poisson("obs", lambda_, observed=count_data)

--------($\lambda_1$)---------$\tau_1$--------($\lambda_2$)------$\tau_2$------($\lambda_3$)-------

import pymc3 as pm
import numpy as np
import matplotlib.pyplot as plt


count_data = np.loadtxt("txtdata.csv")
n_count_data = len(count_data)

with pm.Model() as model:
    alpha = 1.0 / count_data.mean() #1/短信的平均值
    lambda_1 = pm.Exponential("lambda_1", alpha)
    lambda_2 = pm.Exponential("lambda_2", alpha)
    lambda_3= pm.Exponential("lambda_3", alpha)
    tau1 = pm.DiscreteUniform("tau1", lower=0, upper=n_count_data-1) 
    tau2=pm.DiscreteUniform("tau2", lower=tau1, upper=n_count_data)
    
    idx = np.arange(n_count_data)  # Index，当前天数
    lambda_temp = pm.math.switch(tau1 >= idx , lambda_1,lambda_2)
    lambda_ = pm.math.switch(tau2 >= idx,lambda_temp,lambda_3 )


    observation = pm.Poisson("obs", lambda_, observed=count_data)

    trace = pm.sample(10000, tune=2000,cores=1,chains=2)

pm.traceplot(trace)
